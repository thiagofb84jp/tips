Tips for the following tools:


VScode

Intellij

  • CTRL + ALT + L - Permite formatar o código fonte.
  • sout - Exibe a linha System.out.println.
  • ALT + RETURN e CTRL + RETURN - Exibe as dicas.
  • CTRL + SHIFT + R - Executa o programa.

Docker

  • docker ps - Permite a visualização de todos os contâiners criados.
  • docker images - Lista e permite gerenciar todos os contâiners criados.
